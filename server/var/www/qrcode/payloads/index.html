QR Code Payloads

Plain Text (Including URLs)

Encodes any string of text, including URLs.

Examples:

  • Plain text: Hello, World!
  • URL: https://example.com

Use Case: Displays text or redirects to a website.

Email

Encodes an email address or a pre-composed email.

Format:

  • Address only: mailto:example@example.com
  • Pre-filled: mailto:example@example.com?subject=Subject&body=Message

Use Case: Opens the default email client to send an email.

Phone Number

Encodes a phone number.

Format:

  • Phone number: tel:+1234567890

Use Case: Initiates a phone call.

SMS

Encodes a phone number and a text message.

Format:

  • Message: sms:+1234567890?body=Your message here

Use Case: Opens the SMS client with the number and message pre-filled.

WhatsApp Link

Encodes a phone number or a pre-filled WhatsApp message.

Format:

  • Without a message: https://wa.me/1234567890
  • With a pre-filled message: https://wa.me/1234567890?text=Hello+World

Use Case: Opens WhatsApp to chat with the specified number, optionally with a pre-filled message.

Wi-Fi Configuration

Encodes Wi-Fi network details.

Format:

  • WIFI:T:WPA;S:NetworkName;P:Password;;
  • Details:
  • T: Encryption type (WEP, WPA, or empty for no encryption)
  • S: SSID (network name)
  • P: Password

Use Case: Connects to a Wi-Fi network automatically.

Geolocation

Encodes latitude and longitude coordinates.

Format:

  • Coordinates: geo:37.7749,-122.4194

Use Case: Opens the default map application to a specific location.

vCard

Encodes detailed contact information.

Format:

  • Contact information:
    BEGIN:VCARD
    VERSION:3.0
    N:LastName;FirstName
    TEL:+1234567890
    EMAIL:example@example.com
    END:VCARD

Use Case: Saves a contact to the device.

Calendar Event

Encodes event details for scheduling.

Format:

  • Contact information:
    BEGIN:VEVENT
    SUMMARY:Meeting with Team
    DTSTART:20250120T090000
    DTEND:20250120T100000
    LOCATION:Office
    END:VEVENT

Use Case: Adds an event to the device calendar.

Bitcoin and Cryptocurrency Payments

Encodes wallet information and payment details.

Format:

  • Bitcion:bitcoin:1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa?amount=0.01&label=Donation

Use Case: Initiates a cryptocurrency payment.
